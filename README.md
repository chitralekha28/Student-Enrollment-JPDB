Student Enrollment Form using JsonPowerDB
##  Description
This project is a Student Enrollment Form created using **HTML, CSS, JavaScript, and JsonPowerDB (JPDB)**.  
It allows users to **add, update, and manage student records** efficiently using JPDB’s fast and lightweight NoSQL database.
The form captures:
- Roll Number (Primary Key)
- Full Name
- Class
- Birth Date
- Address
- Enrollment Date

This microproject demonstrates the use of JPDB’s RESTful APIs for real-time data handling without a server-side backend.
##  Benefits of Using JsonPowerDB
- **Ultra-fast CRUD operations**
- **Schema-free** — simple to integrate
- **Serverless technology**
- **Supports JSON document storage**
- **Built-in security**
- **Real-time data operations**
- **Saves development time**
- deal for **frontend-only apps**  

JsonPowerDB is perfect for small apps, microprojects, and demo projects that need quick and simple database operations.
